# YouTube Song Searcher
Overview
This is a Python application that provides a graphical user interface (GUI) for searching songs on YouTube.
The user can enter a song or artist name in either Hebrew or English. 

Features
* Search: Uses Selenium and BeautifulSoup to scrape YouTube search results.
* Interface: Built with Tkinter for an easy-to-use desktop experience.
* Playback: Select a result from the list to open and play the video directly in your web browser. 

Technologies Used
* Python
* Selenium & BeautifulSoup4 
* Tkinter (GUI) 
* Webbrowser module
